# 🐢 海龟汤 · 本地推理谜题

一个可在本地打开的中文「海龟汤」（情境推理）游戏网站。题面来自**网络流行题库**与**世界经典情境谜题**，
你可以浏览题库、挑选口味、向**主持人提问**（是 / 不是 / 无关 / 接近），想清楚了再亲手揭晓汤底。

> 深色 / 浅色双主题 · 题库卡片墙 + 解谜提问界面 · 运行后即可体验。

---

## 🚀 两种打开方式

### ① 最简单 —— 双击 `index.html`
直接用浏览器打开，**离线即可浏览与游玩**：
- 自带 **28** 道精选题目（含世界经典谜题），可浏览、看内置提示、揭晓汤底；
- 点「🌐 从网上获取更多」实时联网解锁更多在线题目；
- 想**向主持人提问**（AI 裁判 是/不是/无关/接近）：点右上角 **🔑**，在「浏览器直连」里选 **Anthropic** 或 **Gemini** 并填入对应的 API Key（只存在你本机浏览器、直连各自官方接口），即可在 `file://` 或 GitHub Pages 上使用。Gemini 有免费额度，更适合公开分享。

### ② 完整体验（推荐）—— 启动本地服务器
双击 **`start.command`**，或在终端运行：

```bash
python3 server.py
```

浏览器会自动打开 **http://localhost:8000** 。服务器只依赖 Python 3 标准库，**零安装**。

#### AI 主持人（裁判）—— 本地玩可用多种后端，自动检测、页面里可切换
提问的裁判必须由大模型完成。服务器会**自动检测**你本机可用的后端，右上角 🔑 面板里可随时切换（「自动」会优先用最快的并自动回退）：

| 后端 | 怎么启用 | 说明 |
| --- | --- | --- |
| **Gemini**（REST） | `export GEMINI_API_KEY=AIza...` | 最快，推荐本地玩 |
| **Claude Code CLI** | 已安装并登录 `claude` 即可 | 直接用你的 Claude Code 账号，**无需任何 API Key** |
| **Gemini CLI** | 已安装并登录 `gemini` 即可 | 用 gemini CLI 自己的登录态 |
| **Codex CLI** | 已安装并登录 `codex` 即可 | OpenAI Codex |
| **Anthropic**（REST） | `export ANTHROPIC_API_KEY=sk-ant-...` | 用 Claude 模型 |

例如用 Gemini：

```bash
export GEMINI_API_KEY=AIza...      # 你的 Gemini API Key
python3 server.py
```

- 可选：`TURTLE_SOUP_PROVIDER=gemini|anthropic|gemini-cli|claude-cli|codex-cli`（强制某个后端）、`GEMINI_MODEL`（默认 `gemini-2.5-flash`）、`TURTLE_SOUP_MODEL`（Claude 模型，默认 `claude-opus-4-8`）。
- 页面右上角显示 **🤖 AI 主持** 或 **🔑 设置 Key**，一眼看清当前状态与后端。
- **所有密钥只从环境变量读取，绝不写入任何文件、绝不返回给前端。**

## 🎮 功能一览

| 功能 | 说明 |
| --- | --- |
| 📚 **题库列表** | 卡片墙浏览所有题目，支持**搜索**（标题 / 关键词 / 标签）与**分类**筛选 |
| 🍲 **五种口味** | 清淡 · 推理 · 恐怖 · 温情 · 脑洞，外加 ⭐ 收藏 |
| 🔍 **向主持人提问** | 问任何能用「是 / 不是」回答的问题，AI 主持人给出 是 / 不是 / 是也不是 / 无关 / **接近** / 🎉恭喜（需启用 AI，见上） |
| 🎛️ **选择 AI 后端** | 右上角 🔑 可在 Gemini / Claude Code / Gemini CLI / Codex / Anthropic 之间切换（服务器检测到哪些就有哪些） |
| 💡 **提示（可选）** | 部分题目内置渐进式提示；启用 AI 后还能「求主持人点拨」 |
| 🥄 **公布汤底** | 汤底**默认隐藏**，点击才揭晓——先自己推理！ |
| 🔀 **换一题 / 🎲 随机** | 在当前分类里换下一道 |
| 🌐 **从网上获取更多** | 实时拉取在线题库，去重后并入（离线也保留已获取的题目） |
| 🤖 **AI 出题** | 启用 AI 后，可让大模型**现场原创**一道全新海龟汤（对应原插件的「LLM 自动出题」） |
| 🔢 **提问计数** | 每次提问都会标注「第 N 问」，方便掌握推理节奏 |
| ❓ **玩法帮助** | 首页内置「怎么玩」说明，随时可展开查看规则 |
| ⭐ **收藏 / 进度** | 收藏喜欢的题；自动记录「已看 / 已解」 |
| 🌙 **昼夜主题** | 深色（默认）/ 浅色一键切换 |
| 🔗 **深链** | `?p=<题目ID>` 可直接打开某一题，方便收藏分享 |

快捷键：解谜页按 **Esc** 返回题库。

## 📁 目录结构

```
turtle_soup/
├── index.html            # 主页面（可直接双击打开）
├── css/styles.css        # 样式（深色/浅色主题）
├── js/
│   ├── app.js            # 前端逻辑（题库 / 提问 / 提示 / 汤底）
│   ├── puzzles.js        # 离线自带题库（自动生成）
│   └── webmeta.js        # 在线题库的分类 / 提示 / 数据源（自动生成）
├── server.py             # 本地服务器 + AI 主持人（仅用 Python 标准库）
├── start.command         # macOS 双击启动器
├── tools/build_dataset.py# 题库构建脚本（可重新生成 js/*.js）
└── data/
    ├── _web_source.txt   # 网络题库原始下载
    └── puzzles.json      # 完整题库（参考）
```

重新生成题库（例如更新了 `data/_web_source.txt` 或想调整分类/提示）：

```bash
python3 tools/build_dataset.py
```

## 🔒 隐私与安全
- 提问裁判会把【汤面 + 汤底 + 你的提问】发送给你所选的 AI 后端（Gemini / Anthropic，或本机的 `claude` / `gemini` / `codex` CLI）。**不提问就不会发送任何内容。**
- **所有密钥只从环境变量或你自己的浏览器本地读取，绝不写入仓库、绝不上传到任何第三方服务器**（浏览器直连模式是你的浏览器直接访问 Anthropic 或 Google Gemini 官方接口；GitHub Pages 是公开托管，切勿把任何 Key 写进网页——每位访客用自己的 Key 即可）。
- 浏览、收藏、进度全部保存在本地；「从网上获取更多」只读取公开题库文本。
