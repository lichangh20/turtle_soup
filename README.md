# 🐢 海龟汤 · 本地推理谜题

一个可在本地打开的中文「海龟汤」（情境推理）游戏网站。题面来自**网络流行题库**与**世界经典情境谜题**，
你可以浏览题库、挑选口味、向**主持人提问**（是 / 不是 / 无关 / 接近），想清楚了再亲手揭晓汤底。

> 深色 / 浅色双主题 · 题库卡片墙 + 解谜提问界面 · 运行后即可体验。

---

## 🚀 两种打开方式

### ① 最简单 —— 双击 `index.html`
直接用浏览器打开，**离线即可畅玩**：
- 自带 **28** 道精选题目（含世界经典谜题）；
- 可以「提问」——由**本地裁判**近似判断（无需联网、无需任何配置）；
- 点「🌐 从网上获取更多」还能实时联网解锁更多在线题目。

### ② 完整体验（推荐）—— 启动本地服务器
双击 **`start.command`**，或在终端运行：

```bash
python3 server.py
```

浏览器会自动打开 **http://localhost:8000** 。服务器只依赖 Python 3 标准库，**零安装**。

#### 想让「AI 主持人」精准裁判你的提问？
本项目支持用 Anthropic 的 Claude 模型当主持人（更聪明、能理解语义）。启动前设置密钥即可：

```bash
export ANTHROPIC_API_KEY=sk-ant-xxxxxxxx      # 你的 Anthropic API Key
export TURTLE_SOUP_MODEL=claude-opus-4-8       # 可选，默认就是它；想更快可换 claude-haiku-4-5
python3 server.py
```

- 页面右上角会显示 **🤖 AI 主持** 或 **🧩 本地裁判**，一眼就知道当前模式。
- **没有密钥也完全能玩** —— 会自动回退到「本地裁判（近似）」。

## 🎮 功能一览

| 功能 | 说明 |
| --- | --- |
| 📚 **题库列表** | 卡片墙浏览所有题目，支持**搜索**（标题 / 关键词 / 标签）与**分类**筛选 |
| 🍲 **五种口味** | 清淡 · 推理 · 恐怖 · 温情 · 脑洞，外加 ⭐ 收藏 |
| 🔍 **向主持人提问** | 问任何能用「是 / 不是」回答的问题，得到 是 / 不是 / 是也不是 / 无关 / **接近** / 🎉恭喜 |
| 💡 **提示（可选）** | 部分题目内置渐进式提示；配置了 AI 后还能「求主持人点拨」 |
| 🥄 **公布汤底** | 汤底**默认隐藏**，点击才揭晓——先自己推理！ |
| 🔀 **换一题 / 🎲 随机** | 在当前分类里换下一道 |
| 🌐 **从网上获取更多** | 实时拉取在线题库，去重后并入（离线也保留已获取的题目） |
| 🤖 **AI 出题** | 配置 AI 后，可让 Claude **现场原创**一道全新海龟汤（对应原插件的「LLM 自动出题」） |
| 🔢 **提问计数** | 每次提问都会标注「第 N 问」，方便掌握推理节奏 |
| ❓ **玩法帮助** | 首页内置「怎么玩」说明，随时可展开查看规则 |
| ⭐ **收藏 / 进度** | 收藏喜欢的题；自动记录「已看 / 已解」 |
| 🌙 **昼夜主题** | 深色（默认）/ 浅色一键切换 |
| 🔗 **深链** | `?p=<题目ID>` 可直接打开某一题，方便收藏分享 |

快捷键：解谜页按 **Esc** 返回题库。

## 📁 目录结构

```
turtle_soup/
├── index.html            # 主页面（可直接双击打开）
├── css/styles.css        # 样式（深色/浅色主题）
├── js/
│   ├── app.js            # 前端逻辑（题库 / 提问 / 提示 / 汤底）
│   ├── puzzles.js        # 离线自带题库（自动生成）
│   └── webmeta.js        # 在线题库的分类 / 提示 / 数据源（自动生成）
├── server.py             # 本地服务器 + AI 主持人（仅用 Python 标准库）
├── start.command         # macOS 双击启动器
├── tools/build_dataset.py# 题库构建脚本（可重新生成 js/*.js）
└── data/
    ├── _web_source.txt   # 网络题库原始下载
    └── puzzles.json      # 完整题库（参考）
```

重新生成题库（例如更新了 `data/_web_source.txt` 或想调整分类/提示）：

```bash
python3 tools/build_dataset.py
```

## 🔒 隐私说明
仅当你**自行配置了 `ANTHROPIC_API_KEY`** 并使用「AI 主持人」时，服务器才会把【汤面 + 汤底 + 你的提问】发送到 Anthropic 官方 API 用于裁判。
不配置密钥时，全部逻辑都在你本地运行，不联网（「从网上获取更多」除外，它只读取公开题库文本）。
